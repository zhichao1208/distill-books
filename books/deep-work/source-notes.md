## Chapter → skill field mapping

The book has 2 parts. Part 1 makes the case (why deep work matters, why it's becoming rare, why it should matter to you). Part 2 gives 4 rules. The skill uses Part 2 almost entirely.

### Part 1: The Idea (not used in skill directly, but shapes framing)

| Chapter | How the skill uses it |
|---|---|
| Ch 1: Deep Work Is Valuable | Background only — the skill assumes the user already believes this (they're invoking it) |
| Ch 2: Deep Work Is Rare | Framing for the "shallow expands to fill time" observation in cold-start B |
| Ch 3: Deep Work Is Meaningful | Not used (the skill doesn't try to motivate users philosophically) |

### Part 2: The Rules (load-bearing for skill)

| Chapter / Rule | How the skill uses it |
|---|---|
| **Rule 1: Work Deeply** | The 4 philosophies (Monastic / Bimodal / Rhythmic / Journalistic) are the core method, used in cold-start C and the 4-philosophies table |
| Rule 1 sub: rituals and routines | The "defended by" line in the philosophy output (3 protections) |
| Rule 1 sub: collaboration | Not used in v0.2.3 (collaboration is a separate skill question; out of scope) |
| Rule 1 sub: shutdown rituals | Mentioned in the cant-focus-right-now scene closer (residue + stand up + walk away) |
| **Rule 2: Embrace Boredom** | **Not used as a lens.** It's about training attention generally, which is too abstract for cold-start. Lives in source-notes only. |
| **Rule 3: Quit Social Media** | **Not used as a categorical rule.** The skill's cold-start A includes "phone face down / tabs closed" — which is the same idea applied to *the next block*, not as a lifestyle prescription. Categorical rules don't survive the not_for filter (some users have legitimate work reasons for being on social platforms). |
| **Rule 4: Drain the Shallows** | The "shallow audit" lens, used in cold-start B. Also the framing for the phone+tabs intervention in cold-start A. |

## Concepts deliberately not used (and why)

| Book concept | Why not in skill |
|---|---|
| The Pomodoro Technique (mentioned briefly) | Newport doesn't endorse it strongly; it's not core. The 30-min block in the cant-focus scene happens to overlap but isn't framed as Pomodoro. |
| "Become hard to reach" (full chapter) | Categorical lifestyle prescription. Doesn't survive the not_for filter — many jobs require reachability. |
| "Plan every minute of your day" | Schedule-rigid, fails for users with chaotic days. Replaced by the more flexible "rhythmic block + defended" approach. |
| "Quit social media" as a 30-day digital declutter | This belongs to Newport's later book *Digital Minimalism* (2019), not Deep Work. Out of scope here. |
| Productive meditation | Advanced; assumes the user already has a reliable deep block. Not for first-skill-use. |
| Memorize a deck of cards (training exercise) | Demo of attention training, not a methodology the skill recommends. |

## Chinese edition notes

The Chinese edition (《深度工作》) translates the title literally. Specific term translations the skill keeps in English because the Chinese equivalents lose precision:

| English | Chinese | Why skill keeps English |
|---|---|---|
| Deep work | 深度工作 | Translation works; safe to use either |
| Shallow work | 浮浅工作 | "浮浅" feels slightly judgmental in Chinese; skill uses "shallow" untranslated |
| Attention residue | 注意力残留 | Term is academic in English (Sophie Leroy's research); Chinese phrasing reads stilted |
| Drain the shallows | "排干浮浅事务" | Literal Chinese translation is unwieldy; skill keeps English |
| The 4 philosophies (Monastic / Bimodal / Rhythmic / Journalistic) | 4 种深度哲学 (修道院 / 双峰 / 节奏 / 新闻记者) | Chinese names work; skill mentions both for clarity |

## Related / complementary books

- **A World Without Email** (Newport, 2021) — Same author, narrower scope (workflow / communication). Companion skill candidate for v1.5.
- **Slow Productivity** (Newport, 2024) — Most recent Newport. About sustainable pace + fewer commitments. Could be companion or could replace Deep Work eventually if it proves more durable.
- **Stolen Focus** (Hari, 2022) — Diagnosis (collective + environmental) where Newport is prescription (individual + procedural). Complementary lens; skill planned for v0.5.
- **Hyperfocus** (Bailey, 2018) — Lighter version of the same territory. Skill not planned (Deep Work covers it).
- **Indistractable** (Eyal, 2019) — Adds trigger/temptation analysis. Not planned; Atomic Habits skill already covers habit-level distraction.

## Important limitations / counterpoints

- **The book is written for knowledge workers with some schedule control.** Anyone outside that group (shift workers, customer-facing roles, ER, manual production) does not benefit from this skill — and the `structural_shallow` exit makes this explicit.
- **ADHD critique**: A vocal subset of Newport's readers have noted that "embrace boredom" advice can be harmful for ADHD readers, who often have non-pharmacological need for stimulation that isn't a discipline problem. The skill's `vulnerable_clinical` exit reflects this.
- **The 4 philosophies are descriptive, not prescriptive.** Newport describes them; they're not a forced taxonomy. Some users invent hybrid versions that work. The skill should accommodate, not enforce.
- **The book predates remote work as default (2016).** Some advice (corporate offices, hallway conversations as default shallow work) reads slightly dated. Skill updates it where it matters.
