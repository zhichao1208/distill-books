## Why this book and not adjacent ones

How to Talk is the parenting archetype #2 (alongside The Whole-Brain Child). It's the parent-EVERY-DAY-MOMENTS playbook — the actual words to use in actual conflict moments — which The Whole-Brain Child's brain-level WHY framework needs in order to be operational. The skill exists because the book has been pediatric-office-recommended for 40+ years AND has known failure modes (scripted-feel, neurodivergence non-fit, cultural translation) that the honest-scope skill format directly addresses.

## Public-content basis

This skill was built from public material: Faber/Mazlish official site, the related "How To Talk" group-programs site run by Joanna Faber and Julie King, Goodreads, Penguin Random House book page, third-party Slate review, the academic research on praise / affect labeling that the methods anticipated. No verbatim book text beyond ~50 words.
