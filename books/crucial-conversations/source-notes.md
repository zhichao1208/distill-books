# Crucial Conversations · sources

## Book
- *Crucial Conversations*, Patterson / Grenny / McMillan / Switzler, McGraw-Hill, 2002 (3rd ed 2021)
- ISBN: 978-1260474183 (3rd ed) · Chinese: 《关键对话》, 机械工业出版社, multiple editions
- Sales 5M+ · Goodreads ~4.05
- VitalSmarts (now Crucial Learning) corporate training company sells $1k-$3k/seat workshops

## Chapter / framework mapping
- Start With Heart (your motives, mutual purpose)
- Learn to Look (silence vs violence signals, conversation moving to high-risk)
- Make It Safe (Mutual Purpose + Mutual Respect)
- Master My Stories (separating facts from stories about facts; Path to Action)
- STATE (Share facts, Tell story, Ask others' paths, Talk tentatively, Encourage testing)
- Explore Others' Paths (AMPP: Ask, Mirror, Paraphrase, Prime)
- Move to Action (Who, what, by when, follow up)
- 3rd edition added: inclusive language updates; more workplace-2020s examples

## Critical honesty
- **Corporate-training origin** — book is lead magnet for Crucial Learning's expensive workshops. Some content reads like sales pipeline for that.
- **Acronyms stack fast** — STATE, AMPP, CRIB, ABC, CPR, Fool's Choice, Mutual Purpose, Contrasting, Pool of Shared Meaning. Self-readers can't internalize all of them; skill picks the high-leverage ones (STATE, AMPP, Path to Action).
- **Workplace-shape** — works well for manager-employee, peer-peer, performance feedback. Under-fits intimate relationships, family deep ruptures, parent-teen dynamics where the scripted dialogue feels patronizing or stilted.
- **Path to Action structurally similar to CBT cognitive triangle** — less novel than presented.
- **Assumes good-faith counterpart** — doesn't address abuse, manipulation, NPD, power asymmetry, bad-faith negotiation.
- **East-Asian / hierarchical cultural translation issues** — the direct-dialogue-as-default norm is itself contested in many cultures.

## Concepts deliberately not used
- Specific scripted opener lines that read stilted ("I'd like to share something that's been on my mind...")
- Workshop-context examples that don't translate to self-reader
- Most secondary acronyms (skill focuses on STATE + AMPP)
- Crucial Learning-branded assessments / certification path

## Pairs with
- **Nonviolent Communication (Rosenberg)**: complementary; NVC stronger for intimate relationships, CC stronger for workplace
- **Difficult Conversations** (Stone / Patton / Heen, Harvard Negotiation Project): adjacent territory; more academic
- **Getting to Yes** (Fisher / Ury): negotiation-specific; companion for bargaining contexts
