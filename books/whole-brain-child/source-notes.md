# The Whole-Brain Child · sources

## Book
- *The Whole-Brain Child*, Daniel J. Siegel & Tina Payne Bryson, Delacorte Press, 2011
- ISBN: 978-0553386691 · Chinese: 《全脑教养法》, 浙江人民出版社, 2013
- Siegel: UCLA professor of psychiatry, "Mindsight" framework author
- Bryson: child psychologist, parent educator
- Goodreads ~4.24 / ~60k ratings · ~2M+ copies · NYT bestseller
- Follow-ups: No-Drama Discipline (2014), The Yes Brain (2018), The Power of Showing Up (2020)

## Core methodology
- Integration as the meta-framework: left-right brain + upstairs-downstairs + memory + self + family
- 12 specific strategies, age-tiered:
  - "Name it to tame it" — affect labeling
  - Connect and redirect
  - Engage, don't enrage
  - Move it or lose it (physical movement to shift state)
  - Use the remote of the mind (replay difficult moments)
  - SIFT (Sensations / Images / Feelings / Thoughts)
  - Wheel of awareness
  - Triune brain model (upstairs/downstairs simplification)
- Age-appropriate adaptations (toddler / school-age / teen)

## Critical honesty (LOAD-BEARING)
- **Left-brain / right-brain framing is simplified neuroscience.** Strict hemispheric separation is retired; integration concept useful as metaphor, not literal anatomy.
- **"Upstairs / downstairs" brain** is similarly simplified — useful for parents to picture, technically the prefrontal cortex / limbic system distinction is more complex (and MacLean's "triune brain" model is largely retired in modern neuroscience).
- **Hand model of the brain** — useful teaching tool, simplified science.
- **Strategies work for neurotypical low-acuity moments.** They do NOT substitute for clinical evaluation of ADHD, autism, anxiety, OCD, sensory processing, trauma. Siegel says so explicitly; parents sometimes miss this.
- **Lieberman fMRI on affect-labeling** (which "name it to tame it" rests on) is real but smaller-effect than book implies.

## Concepts deliberately not used
- Strict hemispheric attribution claims (e.g., "your left brain just said...")
- Brain anatomy as literal explanation rather than metaphor
- Replacement-for-clinical-care framing
- Reward/punishment systems (different parenting school; not Siegel)
